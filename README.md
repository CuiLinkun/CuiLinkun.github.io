# 崔林坤的学术个人主页（Cui Linkun's Academic Homepage）
本仓库用于存放我的个人学术主页代码，基于Jekyll模板搭建，部署在GitHub Pages上，访问地址：[https://CuiLinkun.github.io](https://CuiLinkun.github.io)。

## 主页内容
- 个人简介：教育背景、研究方向
- 项目经历：嵌入式系统、智能检测相关项目
- 学术成果：本科毕业论文、课程报告、实习成果
- 荣誉奖项：竞赛获奖、奖学金等

## 技术框架
- 静态网站生成：Jekyll
- 部署平台：GitHub Pages
- 样式：Sass/CSS
